住宿環境

更多詳細資料請自行google或參照更多網友的分享。環島累到忘記拍照，累到有床就很開心很知足了。

以下只會有非常簡短的心得感想。超連結為訂房頁面，絕無業配，純粹當時也是用這平台訂的。

以「水」代表海悅酒店號稱的海景；以「時」代表沐楓商旅的裝潢主題

花蓮海悅酒店

服務普，裝潢典雅老舊。吹風機風力小，窗簾難完全闔上。房內無茶水，要出去走廊裝。

當時住2樓，看不到官網宣稱的海景，也沒有停車場，停車區域在門口或旁邊騎樓。

點我看房價

桃園沐楓商旅

服務極佳，現代風格裝潢。備品是上山採藥。早餐豐盛又美味，也提供膠囊咖啡和電腦。

大推服務品質，很盡力滿足房客特別要求，景色普通但可以看很遠。停車場有分室內外。

旅途紀實

趕路！趕路！趕路！所以比較多街景照

    x Close

    臺東沿海

    風聲不斷襲入安全帽，令雙耳聽不見身旁的海浪，但風神與海神合作交織的進行曲，深深在我心裡譜出此情此景。

  • +
    x Close

    花蓮山路

    幸運地在雨後來訪。路面水分與晴空相照映。清新的空氣、宜人的氣溫，以及在寒冬中卻留有秋意的路樹。

  • +
    x Close

    花蓮崇德隧道

    因為連假和疫情不能出國，即便天氣時冷時陰，在路上很容易能看到也在環島的人，不管是騎車或腳踏車或徒步。

  • +
    x Close

    宜蘭蘇澳

    騎到半路精疲力盡，在蘇花公路的路肩小廟稍作停留恢復體力並伸展筋骨，同時捕捉宜蘭的風景。

  • +
    x Close

    新北北宜公路

    逐步回到熟悉的臺北，與花蓮的山路相比，濕冷又起霧的臺北山景看起來多了一份魔幻鬼魅感。

  • +
    x Close

    新竹香山

    白色的風力發電一柱擎天，強風吹使風車轉動。在浩瀚藍天之下逆風行駛，深感作為人類的渺小。

  • +
    x Close

    彰化鹿港

    人山人海和鬆獅犬。鹿港老街滿溢餅舖和小吃的香氣，連狗也生津垂涎。雖然對我微笑，但應該是期待吃到美食吧！

  • +
    x Close

    彰化二林

    不小心在鹿港待太久，還沒騎回臺南朋友家便日落天黑了。第一天清晨還沒天亮出發，最後一天也以夜路結尾。
